


四点要素

    状态 State
        灵感，创造力，存储小规模问题的结果
    方程 Function
        状态之间的联系，怎么通过小的状态，来算大的状态
    初始化 Intialization
        最极限的小状态是什么, 起点
    答案 Answer
        最大的那个状态是什么，终点


常见四种类型

    Matrix DP (10%)
    Sequence (40%)
    Two Sequences DP (40%)
    Backpack (10%)

    注意点

        贪心算法大多题目靠背答案，所以如果能用动态规划就尽量用动规，不用贪心算法
